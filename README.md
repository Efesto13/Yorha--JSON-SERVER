 # AUTOPARK

Web de reconocimiento de placas para el uso en parkings e integra mercado pago para el procesamiento de pagos.

## FEATURES

- Reconocimiento de Placas
- Panel de Admin
- Integracion de Inteligencia Artificial
- Pasarela de Pagos
- Planes Mensuales
